> ¿Que es Tailwind CSS?
    * Tailwind es un framework CSS con clases preconstruidas que nos permiten estilar elementos HTML de una forma mas rápida.

> Ventajas
    * Agiliza los tiempos de maquetado
    * Utiliza estilos que son estandarizados, para abarcar la mayoria de dispositivos
    * Es adaptable a las necesidades de cada desarrollador
